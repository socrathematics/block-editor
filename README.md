# Course Maker
https://cms.ymath.io/block-editor

## Features
This is a desktop and web app for writing math courses. It features:
* text boxes
* inline math
* display math
* line breaks
* practice problems
* can save your work
* can import work to/from the LaTeXer (practice problems **not** supported cross-application)
